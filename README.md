
# Directives

## Angular.js - UI directives

***

## Build Requirements:

Install UglifyJS & LESS:

```bash
$ [sudo] npm install uglify-js -g
$ [sudo] npm install less -g
```

## Build/Compress:

```bash
$ make build
```
